# 차세대 학원 플랫폼

2025-2026 최신 에듀테크 트렌드를 반영한 AI 기반 차세대 학원 플랫폼

## 🎨 디자인 컨셉

**Natural Intelligence Theme** - 자연적 지성

- **Primary Color**: Mocha Mousse (Pantone 17-1230) - 안정감과 신뢰감을 주는 따뜻한 갈색
- **Accent Color**: Green Aurora (Seoul Color 2025) - 생명력과 긍정적 에너지를 상징하는 초록빛
- **Philosophy**: Functional Minimalism + Liquid Glass Effect

## ✨ 주요 기능

### 1. 개인화 대시보드
- **Bento Grid 레이아웃**: 학습 현황을 한눈에 파악
- **AI 기반 추천**: 개인별 맞춤 학습 콘텐츠 제공
- **게이미피케이션**: 연속 학습일, 배지 획득으로 동기 부여

### 2. 24시간 AI 튜터
- 실시간 질문 응답
- 행정 업무 60% 자동화
- 학습 분석 및 피드백

### 3. LMS (Learning Management System)
- 출석 현황 및 진도율 추적
- 과제 제출 및 성적 관리
- 실시간 학습 데이터 시각화

### 4. 몰입형 강좌 탐색
- Scrollytelling 방식의 콘텐츠 전달
- 마이크로 인터랙션으로 향상된 UX
- 디지털 카탈로그 형태의 강좌 소개

### 5. 다크 모드 2.0
- 주변 조도에 따른 자동 조절
- 눈의 피로 최소화
- 장시간 학습 최적화

## 🛠️ 기술 스택

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **State Management**: React Hooks

## 📦 설치 및 실행

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.

### 빌드
```bash
npm run build
```

### 프로덕션 실행
```bash
npm start
```

## 🎯 핵심 디자인 원칙

### 1. 기능적 미니멀리즘 (Functional Minimalism)
장식적 요소를 배제하고 학습 콘텐츠와 데이터 시각화에 집중

### 2. Liquid Glass Effect
반투명한 유리질감으로 깊이감과 현대적 느낌 표현

### 3. 엄지 친화적 UI (Thumb-friendly)
모바일 최적화된 인터페이스로 언제 어디서나 편리한 학습

### 4. 맥락 인식형 경험 (Context-aware)
사용자의 상태와 학습 패턴을 인식하여 맞춤형 콘텐츠 제공
